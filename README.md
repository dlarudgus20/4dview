# 사용법
**W/A/S/D/마우스** : 3차원 공간의 카메라를 움직입니다.

**R/T** : 4차원 공간의 카메라를 이동시킵니다. (극좌표 theta1)

**F/G** : 4차원 공간의 카메라를 이동시킵니다. (극좌표 theta2)

**V/B** : 4차원 공간의 카메라를 이동시킵니다. (극좌표 theta3)

**Y/U** : 물체를 움직입니다. (XY)

**H/J** : 물체를 움직입니다. (YZ)

**N/M** : 물체를 움직입니다. (ZX)

**I/O** : 물체를 움직입니다. (XW)

**K/L** : 물체를 움직입니다. (YW)

**,/.** : 물체를 움직입니다. (ZW)

**1** : 와이어프레임 모드를 켜거나 끕니다.

**Q**: 종료합니다.

**F1** : 이 도움말을 콘솔에 출력합니다.
